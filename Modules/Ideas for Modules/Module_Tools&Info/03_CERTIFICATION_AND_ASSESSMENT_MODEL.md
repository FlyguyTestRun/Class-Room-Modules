# CoreSkills4AI — Certification Spine & Assessment Model

---

## Certification Levels

### Level 1 — AI Systems Designer
Validated Skills:
- Mental models
- Context engineering
- Failure diagnosis

Assessment:
- Written architecture explanations
- Failure analysis labs

---

### Level 2 — Agent Architect
Validated Skills:
- Agent boundary design
- Multi-agent pattern selection
- Observability instrumentation

Assessment:
- Agent system with enforced constraints
- Telemetry review

---

### Level 3 — AI Cost & Governance Engineer
Validated Skills:
- Cost forecasting
- Governance integration
- Risk-aware architecture

Assessment:
- Cost model
- Governance design review
- Executive-style justification

---

## Assessment Philosophy

- No multiple-choice exams
- Emphasis on explanation, trade-offs, and reasoning
- Students must defend decisions under constraints

---

## Differentiation

CoreSkills4AI certifications signal:
- Systems thinking
- Economic realism
- Production readiness
- Governance literacy
